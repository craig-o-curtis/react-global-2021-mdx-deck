# Takeaways

<Layout>
<>

- Keep learning React
  - Focus on hooks and custom hooks
- Next.js may be worth
- Don't reinvent the wheel
  - Battle-tested npm modules
- Premature optimization = root of evil
- There are no all-knowing wizards of React
- Futureproof yourself
